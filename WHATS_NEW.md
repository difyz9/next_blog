## 🔧 自定义配置

### 修改主题色
```typescript
// tailwind.config.js
theme: {
  extend: {
    colors: {
      primary: {
        50: '#eff6ff',
        500: '#3b82f6',
        600: '#2563eb', // 主色
        700: '#1d4ed8',
      }
    }
  }
}
```

### 调整代码块样式
```css
/* src/app/globals.css */
pre {
  background: #1e293b !important; /* 修改背景色 */
}
```

### 修改动画速度
```css
/* 全局过渡时间 */
.transition-all {
  transition-duration: 200ms; /* 默认 300ms */
}
```
