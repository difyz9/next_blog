## 🎨 设计系统

### 色彩方案
```css
/* 主色调 */
Primary: #2563eb (蓝色)
Secondary: #8b5cf6 (紫色)
Accent: #ec4899 (粉色)

/* 语义色 */
Success: #22c55e (绿色)
Warning: #f59e0b (橙色)
Error: #ef4444 (红色)
Info: #06b6d4 (青色)

/* 中性色 */
Gray: #64748b (灰蓝色)
Background: #ffffff (白色)
Text: #0f172a (深色)
```

### 字体系统
- **标题**: 系统默认 Sans-serif
- **正文**: -apple-system, BlinkMacSystemFont
- **代码**: Fira Code, Courier New
- **字号**: 14px - 48px 阶梯式

### 间距规范
- **基础单位**: 4px
- **常用间距**: 8px, 12px, 16px, 24px, 32px, 48px
- **组件间距**: 16px - 24px
- **章节间距**: 32px - 48px
